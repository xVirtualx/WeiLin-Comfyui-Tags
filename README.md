## 🚀 项目概览

WeiLin-Comfyui-Tags 是基于 **WeiLin-Comfyui-Tools** 生态体系的标签资源优化项目。通过对原标签体系进行逻辑重构、分类升级与内容扩容，为 ComfyUI 生图创作提供更优质、更符合创作直觉的标签支持。

> 💡 **核心理念**：让标签选择从"查找"变为"流畅表达"，提升创作效率与精细度。

## 📥 安装指南

<h4>📋 前提条件</h4>
<p>确保已安装 <a href="https://github.com/weilin9999/WeiLin-Comfyui-Tools">WeiLin-Comfyui-Tools</a>。</p>

<h4>🔧 安装步骤</h4>

<div style="background: #f6f8fa; border-radius: 6px; padding: 16px; margin: 1em 0; border-left: 4px solid #0366d6;">
<ol>
<li><strong>定位目录</strong><br>进入你的 ComfyUI 安装目录：<br><code>ComfyUI/custom_nodes/WeiLin-Comfyui-Tools/user_data/</code></li>
<li><strong>备份原文件</strong><br>重命名或备份原有的 <code>userdatas_zh_CN_tags</code> 文件。</li>
<li><strong>下载新文件</strong><br>从本仓库的 <a href="https://github.com/xVirtualx/WeiLin-Comfyui-Tags/releases">Releases页面</a> 下载最新版标签文件。</li>
<li><strong>替换文件</strong><br>将下载的文件重命名为 <code>userdatas_zh_CN_tags</code> 并放入上述目录。</li>
<li><strong>重启生效</strong><br>重启 ComfyUI 并刷新浏览器页面。</li>
</ol>
</div>
