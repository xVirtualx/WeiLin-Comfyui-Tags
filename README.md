### 📦 安装步骤

确保您已安装 **[WeiLin-Comfyui-Tools](https://github.com/WeiLin-Comfyui-Tools)**。

1.  **进入目录**
    ```bash
    cd ComfyUI/custom_nodes/WeiLin-Comfyui-Tools/user_data/
    ```

2.  **（可选）备份原文件**
    ```bash
    mv userdatas_zh_CN_tags userdatas_zh_CN_tags.backup
    ```

3.  **下载并应用新标签文件**
    - 从本仓库的 **[Releases](https://github.com/xVirtualx/WeiLin-Comfyui-Tags/releases)** 页面下载最新版文件。
    - 将其重命名为 `userdatas_zh_CN_tags` 并放入上述目录。

4.  **重启生效**
    重启 ComfyUI 并刷新浏览器页面，即可体验焕然一新的标签体系。
